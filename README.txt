-> How to Run?

In terminal, navigate to the folder containing 'Tweeter.fsproj' file. 
Execute the commands in following order:
dotnet build 'Tweeter.fsproj'
dotnet run 'Tweeter.fsproj'

This gets server started up. Now from the same project folder run the 'website.html' using any web browser. We have used chrome for demonstration purposes.
